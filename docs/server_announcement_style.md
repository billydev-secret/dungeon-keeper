# The Golden Meadow — announcement & post style guide

A quick reference for mods writing server announcements, event posts, and pinned
messages, so everything reads as one voice. Copy this into a mod-only channel or
pin it. (Draft — tweak the tone/examples to taste.)

## The shape of a post

1. **A one-line headline** in bold — what this is, at a glance.
2. **A short intro line** — the why, in plain language.
3. **Body**, broken into clearly separated sections.
4. **A clear call to action or link** at the end (react here, sign up there,
   read the rules).

## Spacing — the big one

Discord collapses single line breaks visually, so **put a blank line between
every section**. Cramped walls of text are the #1 readability problem.

> **Do**
> ```
> **Movie Night — Friday 8pm**
>
> We're watching a cheesy horror pick chosen by vote.
>
> **How to join**
> React 🍿 below and hop in the Voice Lounge at 8.
>
> **Vote**
> Drop your suggestion in the thread by Thursday.
> ```

> **Don't** — everything jammed together with no blank lines between sections.

## Formatting toolkit

- **Bold** for headings and the one thing you want people to notice. Don't bold
  whole paragraphs — it stops meaning "important."
- Use **`>` block quotes** for rules text, quotes, or an example.
- For anything tabular (times, prices, standings), **line up columns with a code
  block** so they don't wobble:
  ```
  Sat 8pm   Movie Night
  Sun 3pm   Game Day
  Wed 9pm   Risky Rolls
  ```
- **Emoji**: one or two as visual anchors (a headline emoji, a react prompt).
  Not one on every line — it turns into noise.
- **Links & channels**: link the actual channel (`#rules`) or a jump link, don't
  just say "the rules channel." One clear link beats three vague references.

## Voice

- Warm, playful, direct. We're an adults' (21+) community — spicy is fine, but
  an *announcement* should still be skimmable and unambiguous about what to do.
- Say the thing, then stop. If a post needs a "TL;DR," it's probably two posts.

## Before you hit send

- [ ] Blank line between every section?
- [ ] Headline bold and specific?
- [ ] The action/link is obvious and clickable?
- [ ] Read it once on mobile — does it wall-of-text?
